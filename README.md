Projeto elaborado em conjunto com o instrutor da Rocketseat, fazendo parte do plano de ensino. Neste projeto foi criada uma API para controle e manipulação de entregas. Foram usadas bibliotecas como express. Usamos o jwt para autenticação e autorização de usuários, além do uso do ORM Prisma para manipulação do banco de dados. Foi utilizada a biblioteca Zod para realizar validações. Usamos o supertest para realização de testes automatizados e o Docker para criação da imagem do nosso banco de dados. Realizamos também o tratamento de erros, com a estratégia da criação de uma classe e utilização do ZodError, tornando o retorno dos erros mais agradável ao usuário. O projeto foi versionado no VSCODE e as etapas foram comitadas através do gitHub, buscando omitir as informações sensíveis das variáveis de ambiente.
