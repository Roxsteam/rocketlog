Projeto elabora em conjunto com o instrutor da Rocketseat, fazendo parte do plano de ensino. Neste projeto foi criada uma API para controle e manipulação de entregas. Foram usadas bibliotecas como express, jwt. Usamos o ORM prisma para manipalação do banco de dados. Utilizando a biblioteca zod para realizar validações. Usamos o supertest para realização de testes automatizados. Utilizamos também o docker para criação de uma imagem do nosso banco de dados. Realizamos o tratamento de erro com a estratégia da criação de uma classe e da utilização do ZodError, tornando o retorno dos erros mais agradável ao usuário. O projeto foi versionado no VSCODE e as etapas foram comitadas através do gitHub.
